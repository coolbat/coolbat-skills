# Stitch Review Checklist

Review every generated screen against this list before accepting it.

## Strategy Fit

- Is the primary audience obvious within the first screenful?
- Is the primary conversion goal visually dominant?
- Does the page structure match the brief instead of drifting into a generic landing page?
  If not: use Edit Prompt #2 or #4 depending on whether the issue is conversion or trust.

## SEO Fit

- Is there one clear H1 aligned with the intended search theme?
- Do major sections read like real search-intent topics rather than vague brand slogans?
- Are scenario cards or sub-entry points explicit enough to support indexable follow-on pages?
- Does the content feel readable and human rather than stuffed with keywords?
  If SEO structure is weak: use Edit Prompt #1.

## Visual Fit

- Does the palette match the brief?
- Did Stitch introduce a forbidden look, such as purple-blue gradients, dark SaaS styling, perfect three-column sameness, or abstract hero blobs?
- Is there enough hierarchy and asymmetry, or does it look templated?
  If the page looks generic or AI-made: use Edit Prompt #3.

## Resource / Product Fit

- Are cards, modules, or previews describing actual resources rather than generic feature blurbs?
- Can a user tell what is downloadable, previewable, or usable immediately?
- If templates are involved, are duration, audience, materials, or facilitator notes visible where needed?
  If value exchange or practical clarity is weak: use Edit Prompt #2 or #4.

## AI Tool Fit

- If this is an AI tool page, is the workflow concrete rather than just "AI-powered"?
- Can a user tell what they input and what they get back?
- Is there a believable demo, result preview, sample output, or before/after state?
- Is there any trust signal for quality, speed, privacy, or accuracy where relevant?
  If the AI value proposition is vague or the demo is weak: use Edit Prompt #6.

## Copy Quality

- Did Stitch invent placeholder text or generic marketing phrases?
- Are CTA labels concrete?
- Are FAQ questions realistic?
  If the copy is vague, inflated, or product-agnostic: use Edit Prompt #5.

## Iteration Decision

- Keep as-is
- Use Edit Prompt #1 for SEO structure
- Use Edit Prompt #2 for conversion and offer clarity
- Use Edit Prompt #3 for anti-generic visual cleanup
- Use Edit Prompt #4 for buyer trust and credibility
- Use Edit Prompt #5 for copy and messaging rewrite
- Use Edit Prompt #6 for AI tool value clarity and demo specificity
- Regenerate only if multiple categories fail at once
