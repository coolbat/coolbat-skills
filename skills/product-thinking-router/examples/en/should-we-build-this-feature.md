# Example: Should We Build This Feature?

## Problem Diagnosis
This is a discovery + prioritization problem, not only a feature brainstorming problem.

## Recommended Frameworks
- JTBD
- Kano
- RICE

## Why These Frameworks
- JTBD clarifies the real user progress behind the request.
- Kano shows whether the feature is basic, performance, or delight.
- RICE determines whether it deserves investment now.

## Apply to Current Question
Ask:
1. In what situation do users request this?
2. What are they trying to achieve?
3. What do they do today without it?
4. If it is missing, does it create dissatisfaction?
5. How many users would it affect, and at what effort?
