# Example: Define a North Star Metric

## Problem Diagnosis
This is a metrics + strategy alignment problem.

## Recommended Frameworks
- North Star Metric
- HEART

## Why These Frameworks
- North Star Metric aligns the team around delivered user value.
- HEART adds experience guardrails so the team does not optimize value at the expense of usability.

## Apply to Current Question
Ask:
1. What core user value does the product deliver repeatedly?
2. What metric best represents that value being delivered?
3. Which supporting experience signals show whether that value is healthy?
