# Example: Growth Is Stuck

## Problem Diagnosis
This is a growth diagnosis problem. The first job is to locate the broken stage, not invent tactics immediately.

## Recommended Frameworks
- AARRR
- User Journey Map
- 5 Whys

## Why These Frameworks
- AARRR locates the weakest funnel stage.
- User Journey Map reveals friction inside that stage.
- 5 Whys prevents shallow conclusions.

## Apply to Current Question
Check:
1. Is acquisition flat, or is activation weak?
2. At which step do users drop off?
3. What friction exists in onboarding, payment, use, or return flow?
4. What evidence supports the current explanation?

## Misuse Warning
